# Frostpunk Generator Mod

Мод для Minecraft 1.21.1 Fabric по мотивам Frostpunk.

## Установка мода на сервер

1. Скачай `.jar` из GitHub Actions (вкладка Actions → последний билд → Artifacts)
2. Положи в папку `mods/` на сервере
3. Также нужны: **Fabric API**, **Cold Sweat**, **LuckPerms**
4. Перезапусти сервер

## Использование

### Получить блок
```
/give @s frostpunk_generator:control_panel
```

### Поставить блок
Поставь блок `Control Panel` рядом с генератором. Правый клик открывает GUI.

### Роль Инженера (LuckPerms)
Только игроки с группой `engineer` могут открывать панель:
```
/lp user <ник> parent add engineer
```

### Команды (только для операторов)
| Команда | Описание |
|---|---|
| `/generator settemp <°C>` | Установить температуру снаружи |
| `/generator unlockboost` | Разблокировать форсаж |
| `/generator lockboost` | Заблокировать форсаж |
| `/generator repair` | Починить генератор |
| `/generator status` | Статус генератора |
| `/generator log` | Лог событий |

### Пример сюжетного сценария
1. Генератор сломан → случайно ломается пока форсаж заблокирован
2. Игроки чинят → `/generator repair`
3. Оператор разблокирует форсаж → `/generator unlockboost`
4. Форсаж перегружает → при 100% взрыв!

## Текстуры

### Блок панели управления
- Файл: `assets/frostpunk_generator/textures/block/control_panel.png`
- Размер: **16x16 пикселей**
- Нарисуй в Pixilart (pixilart.com) — выбери 16x16 canvas
- После редактирования замени файл в репозитории

### Изображение генератора в GUI
- Файл: `assets/frostpunk_generator/textures/gui/generator_image.png`
- Размер: **130x216 пикселей**
- Открой свою фотографию генератора в Photopea (photopea.com)
- Image → Image Size → 130x216
- File → Export As → PNG
- Замени файл в репозитории

## Сборка через GitHub Actions

1. Залей весь проект на GitHub (новый репозиторий)
2. GitHub автоматически запустит сборку
3. Вкладка **Actions** → выбери последний билд → **Artifacts** → скачай `frostpunk-generator-mod`
4. Внутри архива будет `.jar` файл — это твой мод!
